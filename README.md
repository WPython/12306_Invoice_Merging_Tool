## 功能特性

- 批量添加多个 PDF 发票文件
- 支持 2 张/页 或 4 张/页 布局
- 支持竖向和横向排列
- 可调整页边距
- 实时显示处理进度

## 安装

```bash
pip install PyQt5 PyMuPDF
```

## 使用

```bash
python 12306发票合并工具.py
```

1. 点击"添加发票"选择 PDF 文件
2. 设置每页数量、排列方式、页边距
3. 点击"开始合并并保存"
